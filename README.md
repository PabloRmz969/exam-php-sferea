## Instalación y Uso

### Requisitos
- PHP 8.0 o superior

### Instalación

1. Clona el repositorio:
```bash
git clone https://github.com/PabloRmz969/exam-php-sferea.git
cd exam-php-sferea
```

2. Inicia el servidor de desarrollo:
```bash
php -S localhost:8000 -t public
```

3. Abre tu navegador en:
```
http://localhost:8000
```
